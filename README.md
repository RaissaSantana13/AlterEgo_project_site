# AlterEgo Project Site

Este é o repositório oficial do projeto **AlterEgo_project_site**, inspirado no álbum *Alter Ego* da artista Lisa. O objetivo do site é criar uma experiência interativa, refletindo os conceitos e a estética do álbum.

## Tecnologias Utilizadas
- **HTML, CSS e JavaScript** para a estrutura, estilização e interatividade do site.
